Release Notes

There have been many bug fixes and improvements since last released version 1.7.0 - a major one being that SSHD now consists of several distinct artifact rather than being monolithic, In order to achieve this, some code re-factoring was necessary - especially for SCP, SFTP and using MINA or Netty instead of (the default) NIO2. We therefore decided to name the version as 2.0 in order to emphasize the fact that backward compatibility is not 100% guaranteed.

Please report any feedback to users@mina.apache.org.
